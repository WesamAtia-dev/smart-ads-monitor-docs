\# Smart Ads Monitoring System



\## Overview

Documentation for the Smart Ads Monitoring System, a production-ready project that monitors ads across multiple sources and sends notifications via Telegram.



\## Features

\- Forum/Course Page Detection

\- Multi-source monitoring

\- Telegram notifications

\- Dockerized deployment

\- Dashboard overview with logs and statistics
